# FPTI

**FPTI - Futsal Psychological Type Indicator**

## 설치 및 실행 방법

```
yarn

yarn run dev
```

## 기능

- [x] 진행도 보여주기
- [x] 결과 대로 계산해서 유형분석
- [x] 나와 잘맞는 유형 보여주기
- [x] 공유하기
- [x] 결과 값 잘 나오는지 확인(데이터 검증)
- [x] 공유 할 때 결과값에 맞는 meta 정보 출력(이미지 등) => Next.js로 바꾼 큰 이유
- [x] 카드에 효과 주기
